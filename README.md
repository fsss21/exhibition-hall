# Зал временных выставок

## Краткое описание ПО

Интерактивный киоск зала временных выставок музея «Уткина дача». Содержит историю Уткиной дачи, концепцию и каталог текущей выставки, а также архив прошедших экспозиций и будущих проектов.

## Сценарий работы ПО

### Стартовый экран

Заголовок «ЗАЛ ВРЕМЕННЫХ ВЫСТАВОК». Три блока с кнопкой **Подробнее**:

- **ИСТОРИЯ УТКИНОЙ ДАЧИ** — раздел истории
- **ТЕКУЩАЯ ВЫСТАВКА** — концепция и каталог текущей выставки
- **АРХИВ ВЫСТАВОК** — прошедшие и будущие проекты

### История Уткиной дачи

Линия прогресса с пунктами: хронология, биография владельцев, архитектурные особенности. Текст и галерея фото.  
**Назад** → стартовый экран.  
**Перейти в каталог выставки** → каталог текущей выставки.

### Текущая выставка

Боковое меню:

- **КОНЦЕПЦИЯ** — название и текст выставки (при нескольких выставках — листание)
- **КАТАЛОГ ЭКСПОНАТОВ** — сетка экспонатов → карточка экспоната

**Назад** → стартовый экран.

### Архив выставок

- **ПРОШЕДШИЕ ЭКСПОЗИЦИИ** — галерея прошлых выставок → страница выставки (концепция / каталог экспонатов)
- **БУДУЩИЕ ПРОЕКТЫ** — карточки будущих проектов

С страницы прошедшей выставки **Назад** возвращает в архив.
